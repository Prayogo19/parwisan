# parwisan
> Aplikasi booking bus pariwisata di kota karawang.
***
Aplikasi ini dibangun menggunakan `HTML`,`CSS`,`PHP Native`, `Javascript`.

Daftar Halaman
1. Admin
  * Dashboard
  * CRUD Daftar Bus
  * Konfirmasi Booking
  * Checking Pelunasan
2. User
  * Profil
  * Riwayat Booking
  * Form Booking
  * Daftar Bus
